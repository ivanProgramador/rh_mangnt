Remoção de registros 
  A remoção de registros do sistema vaiu usar uma estrategia de Soft delete isso pode ajudar a evitar acidentes, 
  então ao ives de apagar a linha do banco de dados eu vou soemnte presncehr o campo deleted_at com a data atual 
   
para ativar o SoftDelete tem que ir no model e avisar que deseja essa função como eu fiz abaixo 
com isso o sistema não vai mais apagara a linha e ssim preccher data/hora do campo deleted_at. 

class User extends Authenticable
{
    use HasFactory;
    use Notifiable;
    use SoftDeletes;

    //ligando o model usuario ao model de details 

      public function detail()
        {
           return $this->hasOne(UserDetail::class);
         }
      public function department()
         {
            return $this->belongsTo(Department::class);
         }

}
